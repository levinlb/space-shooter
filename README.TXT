------------------------------------------------------------------------
SPACE SHOOTER
------------------------------------------------------------------------

PROJECT TITLE:
  Space Shooter (Greenfoot)

PURPOSE OF PROJECT:
  Ein wellenbasierter 2D-Weltraum-Shooter. Alle Grafiken werden im Code als
  Pixel-Art erzeugt - es werden keine Bilddateien benoetigt.

VERSION or DATE:
  2026

HOW TO START THIS PROJECT:
  Projekt in Greenfoot oeffnen, MyWorld erzeugen und auf "Run" klicken.

AUTHORS:
  Space-Shooter-Projekt

USER INSTRUCTIONS:

  Steuerung
  ---------
    Pfeiltasten  - Schiff bewegen
    Leertaste    - Schiessen (Hauptkanone)
    Shift        - Boost (verbraucht die Boost-Leiste)
    X            - Nova-Bombe ausloesen (wenn die Waffe vorhanden ist)
    1-4          - Im Shop: Upgrade kaufen
    1-3          - Bei der Waffenwahl: Waffe waehlen
    ENTER        - Naechste Welle starten
    R            - Nach Game Over neu starten

  Spielablauf
  -----------
    Gegner kommen in Wellen. Nach jeder Welle oeffnet sich der Upgrade-Shop
    (Waffe, Panzerung, Antrieb, Turret). Auf jeder 5. Welle erscheint ein
    BOSS mit mehreren Angriffsmustern und zwei Phasen. Nach dem Sieg ueber
    einen Boss darf man aus 3 zufaelligen Waffen eine waehlen - besitzt man
    sie schon, wird sie staerker (Brotato-Prinzip).

  Gegnertypen
  -----------
    Alien     - treibender Asteroid
    Attacker  - Jaeger, der gezielt schiesst
    Drohne    - schnell, fliegt in Schlangenlinien
    Rammer    - zielt und stuerzt sich dann auf das Schiff
    Teiler    - zerfaellt beim Tod in zwei kleine Splitter
    Bomber    - schwer gepanzert, feuert Streufaecher
    Boss      - Kreuzer / Zerstoerer / Mutterschiff

  Draft-Waffen (nach Bossen)
  --------------------------
    Streuschuss   - Faecher aus Schuessen
    Lenkraketen   - suchende Raketen
    Laserstrahl   - durchschlagender Strahl
    Plasma        - grosse, durchdringende Plasmakugel
    Kampfdrohne   - umkreisende Begleitdrohne(n)
    Nova-Bombe    - aktive Faehigkeit (Taste X): Rundumschlag + raeumt Geschosse
